# Home automation

## Installation

1. Clone / form repository
2. Create a `.env` file in your project root with the following variables:
    - HUE_HOST
    - HUE_USER
3. Run `npm install`
4. Run `node app.js`

## Supported controllers

* Phillips HUE (color E27 only)
